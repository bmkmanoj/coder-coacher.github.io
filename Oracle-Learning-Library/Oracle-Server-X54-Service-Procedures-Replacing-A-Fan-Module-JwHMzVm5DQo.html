Oracle Server X5-4 Service Procedures: Replacing A Fan Module

2015-09-09
Oracle Server X5-4 Service Procedures Replacing a Fan Module When a fan module fails, the system lights its Service Action Required indicators. When this happens, use Oracle ILOM to verify the fault and turn on the server’s Locator indicator. The Locator indicator blinks to help you find the server. To identify the failed fan module, look for its lit fault indicator. To remove the fan module, lift its handle and pull it straight up and out of the server. Caution: Possible over-temperature condition. Install the replacement fan module within 60 seconds. To install the replacement fan module, align it in the slot. The fan module is keyed and can only be installed one way. Carefully, lower the fan module into the server.To latch it, press at the point indicated on the label. Ensure that the green fan module OK indicator is lit. For complete service procedures, see your Service Manual on www.oracle.com/goto/x5-4/docs-videos
