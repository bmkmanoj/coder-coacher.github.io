MAX Designer - User Interface Tour

2016-05-05
The MAX Designer gives you everything you need to build your UI, hook up your data, test your app and publish it. The Designer has three main areas: The Navbar, which has tools for accessing the UI components, exploring data and managing your screens. The Preview, which lets you see your work in progress and select components that you configure using the third area of the Designer, the Properties Inspector. You'll form your screens by dragging UI widgets from the Component Palette to the Preview. Then, you'll use the Properties Inspector to configure how the screens, and the UI components within them appear, use data, and behave. You can also access the QuickStart wizards, which prompt you through adding data, creating lists with drill downs, and creating interactive forms. As you click the breadcrumb trail, or within the Preview, you'll notice how the configuration options in the Properties Inspector change to reflect your current selection. At any point, you can preview your app to see how it displays live data and responds to user input. When you're done, you can return to the Designer to continue your work. When you're ready to make your app available to your users, you can quickly build it and then distribute it using the generated QR code. Thanks for watching!
