Oracle Developer Cloud Service: Performing Code Reviews

2016-03-04
use the merge request tab to request code reviews and to merge changes from one branch to another the list of merge request can be filtered using predefined standard searches when creating a new request specify the target and review branches along with the project members you want to review the request there can be only one active request per branch only branches with no active requests are displayed in the list of available review branches for each request there are tabs for viewing changes and leaving comments in the conversation tab you can post general comments about the request as well as comments about specific changes to the code other reviewers can see the comments and post the rain the commits tab shows a list of commits made to the review branch and the files that were changed to changed files tab provides details of the changes made to each file and reviewer comments are displayed in line when adding a comment select publish to make it visible to others in the change files tab and the conversation tab in the conversation tab reviewers can approve or reject proposed changes after code changes are reviewed you can merge the changes from the review branch to the target branch merge request changes in comments are displayed in activities feed on the home page for more information see the Oracle developer cloud service documentation page
