Oracle Social Network: Annotating a Document (Release 8)

2013-08-12
welcome to the Oracle social network video series this video covers annotating documents and viewing those annotations an annotation is a comment added to a document an annotation might be a review comment provide further explanation or ask a question about the document you can access the annotation in two ways by opening the document from the documents page or by opening the document from the conversation you can use annotation filters to limit your view to just the annotations you want to see there are several tools available for annotating a document they include a highlighter a pushpin and ellipse a pen the Select icon and a rectangle clicking the publish button posts all of your new annotations both on the document and in the conversation itself but if you're not ready to publish it you can simply save your unpublished comments then come back later and click publish thank you for watching for more information be sure to visit cloud.oracle.com
