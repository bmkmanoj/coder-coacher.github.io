Printing Dashboard Pages Using Oracle BI EE

2015-05-13
printing dashboard pages using Oracle bi I'm a sales manager and I am visiting one of our best customers at their headquarters tomorrow this customer site is secure and as such I can't take my laptop or mobile device inside their facility I have decided that I want to take along a copy of their order details which will show our customer how well we have been meeting their order requirements hopefully this will stimulate more and larger orders in the future I see that the sales performance dashboard is listed on the home page in the dashboards area I click the sales performance link the sales performance dashboard fulfillment page is displayed I want to print the fulfillment page that includes the order detail analysis I click the page options button and then I select print I see that I have two choices printable PDF or printable HTML I select printable PDF the fulfillment page with the order detail analysis opens in an Adobe Acrobat PDF file viewer I review the content and then I use the Print option in the file viewer to print the file now that I have the printed content for my customers order details I feel confident that I can present a compelling story to my customer indicating how well we have fulfilled their requirements this will provide me with the impetus that I need to ask for larger and additional orders which will net more revenue for my team thanks for watching
