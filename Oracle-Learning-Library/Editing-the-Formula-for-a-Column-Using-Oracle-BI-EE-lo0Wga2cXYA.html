editing the formula for a column using Oracle bi I am a sales manager and I want to create a target revenue column in the brand revenue analysis first I must find the brand revenue analysis I select analysis in the search list and then I enter brand Rev in the search field I click the search button the analysis is displayed in the catalog area I click Edit I need to add a new revenue column and then change the formula I click the criteria tab I expand the revenue metrics folder and then I double-click revenue to add it to the analysis I click the options button for the new revenue column and then I select edit formula I want the target revenue to be seven percent higher than revenue I click in the column formula box and then I click the multiply operator in the tool bar at the bottom I click again in the column formula box and then I enter 1.07 immediately following the multiply operator to multiply the column by seven percent I want to change the name of the column so I select the custom headings checkbox I enter target reve reve plus seven percent in the column headings field and then I click OK to save the column formula I want to see the results after adding the new column so I click the results tab I need to save the updated analysis I click Save I've updated the brand revenue analysis with a new calculation now that I've created this calculation I'll be better able to develop my team's goals and evaluate their contributions to our revenue projections thanks for watching
