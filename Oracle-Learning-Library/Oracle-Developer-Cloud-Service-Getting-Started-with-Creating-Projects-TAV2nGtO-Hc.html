Oracle Developer Cloud Service: Getting Started with Creating Projects

2015-03-11
an Oracle developer cloud service project is a collection of tools and services that help developers share resources and project information when they collaborate on a project each project can have one maven repository and multiple git repositories for managing the source code each project also comes with an issue tracker a build server and wiki server for internal documentation before you create a project consider how you expect it to develop do you want to have one project with many contributors working from a single source file repository or will the project be composed of several smaller independent projects and teams do you want to use one issue tracker and a wiki for your project the architecture of your project will influence how you'll want to set up the project on the Oracle developer cloud service when you create a project and specify the details choose a privacy setting to control who can see the project if you select private only project members can see the project share projects are visible to all organization members after you finish the wizard and your project is created select the organization members that you want to be project members the identity domain administrator must add the members to the organization before they can be added to a project a project member can be assigned a specific role however the role only applies to that project on the administration page the project owner can modify the project properties and specify additional settings
