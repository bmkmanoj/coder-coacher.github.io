by using enterprise manager cloud control log in to enterprise manager cloud control once logged in use the Enterprise menu to navigate to the middleware provisioning screen under middleware provisioning deployment procedures select deploy undeploy Java EE applications and click launch select the operation desired in this demo I am choosing to redeploy an application under targets click the plus sign to select a WebLogic domain in the pop-up window select a domain and click the select button when the domain appears in the targets list since this is a redeployment find the application under the server on which it is already deployed and select it then click Next select the application and click Edit in the pop-up window you need to find a version of the application that has already been placed in the software library enter a component name and click the search button once the application is listed select the version you wish to redeploy and click the arrow button to move it to the components selected for deployments list on this window you may also set the application start mode and retirement policy which I shall leave it there defaults then click OK notice the warning that the application is not versioned I know that so I will just click OK to close this dialog box then click Next select the domain to enter its credentials the domain credentials are the administration level username and password for the admin server of the domain we have already saved these as preferred credentials which are selected by default then click the admin server under the domain to enter its credentials these credentials are for the host machine where the admin server is running again we have already saved preferred credentials for this host so I will leave the preferred credentials selected once both credentials are set click Next give the job a name I'm going to call it redeploy benefits app choose when to start the job I am choosing immediately you may also select whether or not you wish a grace period if it cannot start at exactly the time you would like and pick a maximum duration for the job then click Next on the review screen review your choices and then click redeploy on the procedure activity tab of the deployment procedure manager screen you can select how often to automatically update the screen from the view data drop-down list
