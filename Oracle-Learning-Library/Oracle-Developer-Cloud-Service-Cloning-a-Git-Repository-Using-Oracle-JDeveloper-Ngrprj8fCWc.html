Oracle Developer Cloud Service: Cloning a Git Repository Using Oracle JDeveloper

2015-03-16
you can use Oracle JDeveloper to clone the get repositories of your Oracle developer cloud service projects the URL of the git repository is visible on the project home page in JDeveloper use the clone from git wizard to create a local clone of the git repository to use the wizard the remote repository must be initialized and contain at least one branch and the wizard you specify the location and name of your local clone after you create the local clone of the repository you're prompted to create a JDeveloper project through the sources if no project exists if the repository contains a JDeveloper project or you create a new project the project is visible in the application navigator the project sources are now under version control in the application navigator new or modified files are badged to show that the changes need to be added or committed use the pending changes window to view a list of the change files waiting to be committed you can select the files that you want to include in the commit after the changes are committed to the local clone use the push to get wizard to push the changes to the repository of the Oracle developer cloud service project on the project homepage reload the page to see the commit message in the list of the latest activities the link to the commit opens a page in the code tab that shows a diff of the changes made in the commit for more information on Oracle developer cloud service see the Oracle developer cloud service documentation page you
