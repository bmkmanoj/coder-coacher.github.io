5 Minutes with Primavera Unifier - Adding Summary Elements from Query Tabs

2015-04-30
Hi, this is Aloka from Oracle Primavera. Today you will see how to define and add summary elements to the upper forms of contracts from the Change Orders query tab in Primavera Unifier. Summary elements allow you to view summarized information from query tabs without having to open individual records in the tabs to compare, calculate, or analyze the information. Let's examine a contract. It has two query tabs that display records referencing this contract. There are several numerical fields in the Change Order's upper form. We want to see the schedule and cost impact on the contract scope at a glance without opening any record in the query tab. Adding summary elements to the Contract upper form will provide that information. So navigate to uDesigner. Navigate to Contracts. click Query Items Summary Elements Note that some summary elements are already created to expedite the demonstration. Click New to define a new summary element. Enter an Element Name, for example: "Schedule_impact" Entered a Form Label. Next, select a Summary Type. The options available are: Average, Count, Total, Maximum, and Minimum. The interface changes dynamically for the type you select. For example, if you select Total, you have Query Tab, Data Element, and Data Definition Types to be filled in. If you select Count, then only Query Tab needs to be selected. Let's select Total. In the Query Tab list, selected the query tab from which you want to derive the information. The Data Element list displays fields based on currency, decimal, and integer amount data definitions in the upper form of the Change Orders. Select "Schedule Impact (Days)." Note that Data Definition Type is automatically populated. Optionally, add a description and click OK. Next, you will add these elements to the upper form. Click Upper Forms. Open any form. We have already added a block with the summary elements. Click an empty cell. Then click Summary Select "Total Schedule Impact" And click OK. Save the form. And close the window. Follow the same steps to add the summary elements to the other upper forms Mark Contracts as complete. And then click Deploy to deploy the form. Return to your project and navigate to the Contracts log. Expand the upper form. Note the values are reflected. Total Schedule Impact on this contract scope is by 40 days and The Total Amount of the Change Orders is $850. Thank you for watching "Adding Summary Elements from Query Tabs."
