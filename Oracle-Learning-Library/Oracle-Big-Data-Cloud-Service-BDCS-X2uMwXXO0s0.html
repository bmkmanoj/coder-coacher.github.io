Oracle Big Data Cloud Service (BDCS)

2016-11-09
Oracle Big Data cloud service brings you the power of Hadoop delivered as a secure automated elastic service which is dedicated and fully managed by Oracle the welcome email you received has links to the resources to help you get the most out of Oracle Big Data cloud service as well as details about accessing your account provide your identity domain and proceed to the login screen provide the username and password to sign in the Oracle Big Data cloud service dashboard lists all of your instances and provides high-level information about each instance when creating an instance configure the details of your service such as a name description your Big Data appliance system your SSH key cloudera administrator password the Kerberos and the username and password of your Oracle storage cloud service you can skip this step if you don't have an account or if the Oracle storage cloud service is not required for this specific service once you are satisfied with the information you are ready to provision the instance when you return to the Oracle Big Data cloud service console you will see your new instance is being created thanks for watching
