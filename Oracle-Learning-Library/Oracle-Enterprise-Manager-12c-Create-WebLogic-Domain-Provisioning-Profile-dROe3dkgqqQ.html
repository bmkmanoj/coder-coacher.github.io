Oracle Enterprise Manager 12c: Create WebLogic Domain Provisioning Profile

2011-09-21
welcome to the Oracle Enterprise Manager Cloud Control 12c demo series in this demonstration I will show you how to create a middleware provisioning profile once you are logged in to enterprise manager Cloud Control as a designer and have navigated to the middleware provisioning screen under middleware provisioning profiles click create select the folder where you want the profile to reside then select actions create entity component in the create entity pop-up use the drop-down list to select WebLogic domain provisioning profile then click continue give the profile a name and optional description also optional are the product information and adding attachments and notes when finished with these things click Next to select the domain click the magnifying glass popupwindow find the domain in the list select it and click select on the configure screen you can change the working directory if desired I will leave it at its default information about the source domain is listed the host credentials must be set we have previously set up some preferred credentials which are selected by default in which i will use when the working directory and hos credentials are set click Next on the review screen if everything looks right click save and upload on the software library screen you can see the message that the profile was successfully created and if you look in the folder we selected for it you can see the profile itself in the library so that is how to create a provisioning profile it can be used for cloning our gold image domain at any time thanks for watching this demonstration
