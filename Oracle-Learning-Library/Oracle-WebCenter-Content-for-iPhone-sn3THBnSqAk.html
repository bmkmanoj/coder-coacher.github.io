Oracle WebCenter Content for iPhone

2014-07-11
oracle webcenter content for iphone enables users to manage content from their iphone in this video you will learn how to use the webcenter content for iphone app to connect to the web center content server search for you share and download content mark items as favorites to connect to oracle webcenter content server you will need your username password and the URL for the content server when you open the application all libraries available to you are displayed there are enterprise and system libraries as well as folders browse through the libraries and folders created within oracle webcenter content server to find content or if you're not certain where the content item is located you can use the search feature the content item can be viewed marked as a favorite shared either as a link or as an email attachment and download it to your iPhone for offline use the content item can also be opened in another program or printed selecting the information icon displays the metadata information for the content item or folder you are viewing from the menu bar you can quickly access the items you have previously marked as favorites likewise you can quickly access the items you have previously downloaded on your iPhone for offline use view them and delete them locally thank you for watching for more information please visit oracle com / web center and be sure to join in the conversation
