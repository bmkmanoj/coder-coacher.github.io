Java SE Applications for Oracle Application Container Cloud

2016-01-05
oracle application container cloud provides a complete solution for deploying java standard edition applications in the oracle cloud you can build self-contained java SE applications using any commercial or open source framework including spring play jai thon jruby jersey and drop wizard applications can be deployed executed and scaled up in the oracle application container cloud cloud applications typically expose web applications rest services or WebSocket applications using embedded servers like tomcat jetty glassfish or grizzly in addition oracle application container cloud integrates with other Oracle cloud services such as database cloud service storage cloud service messaging cloud service and Java cloud service java SE applications run on Oracle Linux in docker containers when you deploy your application and Oracle application container cloud a container is dynamically generated for your application the implication is that any application that runs in this environment must read key configuration information like the hostname and port number from the environment before the application starts to deploy an application oracle application container cloud it's packaged in a zip or tgz file that contains all required jars class files and resources along with the manifest JSON files the manifest JSON file provides metadata about your java application this information includes the version of Java and the command used to execute the application java SE applications can be deployed through a user interface or by using a REST API java SE is ideal for building lightweight applications and micro services oracle application container cloud provides the runtime environment to deploy those applications in the cloud thank you for watching for more information about how to create and deploy java SE applications see the series of tutorials in the Oracle help center and Oracle Learning Library
