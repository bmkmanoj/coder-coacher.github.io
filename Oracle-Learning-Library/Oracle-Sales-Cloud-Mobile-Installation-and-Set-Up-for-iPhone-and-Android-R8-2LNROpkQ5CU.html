Oracle Sales Cloud Mobile Installation and Set Up for iPhone and Android R8

2014-07-06
hello my name is Ken in this tutorial I'll show you how to install the Oracle sales cloud mobile application for Android and iPhones you can install the Oracle sales cloud mobile application from the Apple App Store or androids google play when the installation is done open the application to complete the initial setup if you do not have your sign-in information yet use the demo mode to see what the application is all about the first time the application opens you'll need to enter your host the host is specific to your company instance and must be provided by your company administrator copy and paste the server name into the host field then enter your Oracle sales cloud username and password you don't need to go in advanced settings for a typical installation however you can go into the advanced settings page to find your application version and build information once you've entered all the necessary information tap the Sign In button now you're signed in and ready to use the application you can find additional learning resources in your Oracle Fusion help application as well as in the Oracle Learning Library and at Oracle University thank you
