How do I create a new report?

2013-08-19
welcome to the tutorial how do I create a new report I'm Donna Hoban principal technical writer with the Oracle business intelligence documentation team and I will be taking you through the tutorial to begin from the Navigator menu select reports and analytics the reports and analytics page is displayed then click create select analysis and select a subject area for this demo we will select the sales CRM pipeline subject area the next step is to select the columns to include in the analysis first expand the subject area folder and for each comment you want to add select the column and the subject areas let's click Add to move it to the selected columns list for this demo we want to create an analysis that shows opportunities by sales stage therefore we will add five columns from the subject areas lists and move them to the selected columns list the first column to add is the employee name column from the employee folder then the opportunity name sales stage name and status columns from the opportunity folder and finally the number of opportunities from the pipeline facts folder then click Next the next step is to specify the views to include in the analysis and to enter a title views of the ways in which to visually present the data for this demo we will select a bar graph and enter opportunity count by sales stage for the title then click Next optionally we can edit the graph and sort and filter the analysis for this demo we will skip these optional steps by clicking next twice the last step is to save the analysis by specifying the analysis name and selecting the folder in which to save it for this demo we will enter opportunities by sales stage as the name and select the all opportunities folder then click Submit and confirm the save I'm Donna Hoban thanks for watching
