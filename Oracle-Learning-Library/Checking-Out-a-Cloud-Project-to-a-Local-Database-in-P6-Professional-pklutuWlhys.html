Checking Out a Cloud Project to a Local Database in P6 Professional

2017-03-08
p6 professional 17.3 allows you to check out projects from the cloud to a local database this feature enables you to work on projects when not connected to the cloud checking out the project may also lead to better performance for data intensive projects in this video we'll explore the feature starting in p6 professional logged into a cloud database initiate the checkout process and then select open projects you want to check out here's the option to check out to a local sequel Lite database sequel Lite is embedded within p6 professional and does not require additional installation or setup each time you check out from a cloud database a corresponding local database is created as the warning indicates this may overwrite a previously created local database any checked out projects are also displayed in the warning message enter a password for the new local database when the check out process is complete exit out of p6 professional next login to the newly created local database open the project and work on it locally let's add an activity to the project when you are finished your work log back into the cloud database check the project back in when this process is complete the cloud database is updated with your changes
