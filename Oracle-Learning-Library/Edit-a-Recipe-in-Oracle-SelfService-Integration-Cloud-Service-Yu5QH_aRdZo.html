Edit a Recipe in Oracle Self-Service Integration Cloud Service

2018-03-30
from your dashboard click a recipe to open it in the recipe editor the recipe editor shows you hints for which fields require data you can click each message and the editor moves to that section starting in the name and description section you can view the data and make any desired changes all recipes require a name the trigger determines when the recipe will run since this is a public recipe the trigger app name is already listed all that you need to do is authorize an account for the application once your account is authorized the rest of the trigger fields are populated if you are creating a new recipe then you need to add data to the fields the actions section defines the actions the recipe will take when it runs authorize an account and then review and enter data as required you can map data to an actions form fields by dragging and dropping items available in the cloud app data section you can add multiple actions to further define the data you want to gather for more information see cloud.oracle.com / self-service integration
