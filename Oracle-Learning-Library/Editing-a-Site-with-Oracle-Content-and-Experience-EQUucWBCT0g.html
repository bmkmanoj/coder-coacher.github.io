Editing a Site with Oracle Content and Experience

2017-12-14
hi and welcome Oracle content and experience cloud provides a simple way to create and manage your Enterprise focused websites when you first sign in you see a list of all sites to which you have access to edit a site right-click the site and select edit from the menu any edits you make are stored in an update until you are ready to publish them you can choose to make your edits in an existing update or create a new one this way if different people are responsible for different areas of the site multiple updates can be edited at the same time for example you can make any number of changes to a site in any number of updates and time them for a coordinated release to support a sales promotion product launch or other enterprise wide event the Edit toolbar allows you to switch between different updates easily preview the site in a new browser window preview how the site looks in different resolutions and orientations and Save Changes until you're ready to publish the update when you're ready to edit a site the edits which enables a side panel with various options when creating a new page it can be just a link in the site navigation to an external page or a web page design in Oracle sites cloud service where you specify the name layout title and other pertinent information organizing your site navigation is as simple as drag and drop as is adding components such as buttons documents images or other items just drag the component to a place on the page where you want it an icon shows you where the component will be placed use the menu to adjust the settings you can even add third-party components or components you have created yourself if you want to change the look of a page select from predefined themes that you create or others have shared with you when you're done making your edits save the update and if you're ready to merge the update with your base site click publish thanks for watching for more information visit cloud.oracle.com
