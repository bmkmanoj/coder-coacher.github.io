Sun Server X4-8: Replacing a Power Supply

2015-04-15
Sun Server X4-8 Service Procedures: Replacing a Power Supply Important For complete service procedures, see the Service Manual. See the link at the end of this video. The Sun Server X4-8 includes 4 hot-pluggable 220-240 volt power supply units with N+N redundancy. With N+N redundancy, two power supplies are used and two are standby. When a power supply fails, the system alerts you by logging an event in the system event log. And lighting the power supply Service Action Required indicator. Procedure Overview To replace a power supply, verify the power supply fault using Oracle ILOM. Identify the failed component using the fault indicator. Replace the component. To verify a power supply fault, log into Oracle ILOM and check the status of the power subsystem from the Summary screen. The status section provides the current state of all the subsystems. Identify the failed component and record its designation. Before going to the server, use Oracle ILOM to turn on the server's Locator indicator. The Locator indicator helps you find the server. The power supply units are hot-service components, so you do not need to power off the server to replace a power supply. At the server, press the Locator indicator button to turn it off. To identify the failed power supply, look for its lit fault indicator. To remove the power supply, unlock the power supply lever by squeezing together the two green release latches. Rotate the lever to the open position. This action disengages the power supply. Using two hands, slowly slide it completely out of the server. To install the replacement power supply, rotate the locking lever to the open position. Align the power supply with the slot and slide it in until it stops. Rotate the locking lever to its closed and locked position. This action fully inserts and powers-on the power supply. The power OK indicator lights. Ensure that the service action required indicator is not lit. Summary To maintain N+N redundancy replace a power supply as soon as it fails. Verify the power supply fault using Oracle ILOM. Power supplies are hot-pluggable. You do not need to power off the server to replace one. For complete service procedures, see your Service Manual available at: www.oracle.com/goto/X4-8/docs
