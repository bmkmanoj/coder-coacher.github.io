Loading Data from Relational Sources Using Data Sync

2015-12-15
do you need a way to upload customer fulfillment and revenue details from your on-premises database to a provisioned cloud service now you can with Data Sync begin by searching for the data source and selecting the appropriate tables then import the tables to upload by default all table attributes are copied and the entries that display in the target tables tab are created automatically when you import the source inspect the column attributes for the customers table if the table contains columns that are not needed for analysis select the inactive checkbox for the specific columns because these tables will be loaded incrementally you'll need to define a load strategy for each table select a user key and a date timestamp column that uniquely identifies each record for incremental update define the load strategy for the other tables to save the changes now you know how to upload a relational data source and define an incremental load strategy for a relational source using Data Sync thanks for watching you
