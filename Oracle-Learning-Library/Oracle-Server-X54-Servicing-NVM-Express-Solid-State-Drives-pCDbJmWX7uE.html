Oracle Server X5-4: Servicing NVM Express Solid State Drives

2015-09-09
Oracle Server X5-4 Servicing NVM Express Solid State Drives. This video describes the Oracle Server X5-4 NVM express solid state drive option and hot plug replacement. NVM express, or non-volatile memory express, is a specification that includes a command and feature set designed for PCI express solid state drives called "SSDs." NVMe SSDs are supported with Oracle Solaris or Oracle Linux operating systems. NVMe support is configured at the factory as an option. This includes the installation of a PCIe NVMe switch card in the server's PCIe slot 1. Note: The PCIe NVMe switch card does not work in any other slot. An NVMe cable connects the card to the disk backplane. Up to four NVMe SSD drives are supported and are customer installable. Drive slots two through five support either SAS or NVMe drives. NVMe SSDs are not supported in drive slots 0 or 1. Do not use NVMe SSDs as host boot drives. NVMe SSDs are hot pluggable. You do not need to power off the system to replace one. Caution: Surprise hot removal of an NVMe SDD can cause data loss and crash the operating system. For instructions on safely hot plugging an NVMe SSD, see the Service Manual. To enable NVMe SSD hot plug, configure support in the operating system. For Oracle Solaris, the hot plug service must be running. For Oracle Linux, a pci kernel boot argument must be set. Before hot plugging an NVMe SSD, identify its virtual PCIe slot number. Starting with the drive slot labeled "NVMe0", slots are numbered from 100 to 103. To hot plug a drive, obtain the virtual PCIe slot number as seen by the operating system. For Oracle Solaris, use the hotplug list command. For Oracle Linux, use egrep with the -H option. After quiesing activity and unmounting the drive, power off its virtual PCIe slot using operating system commands. For Oracle Solaris, use hotplug poweroff. For Oracle Linux, use echo zero. Once the blue "Ready to Remove" indicator lights, remove the drive. Install the new drive. After installing the drive, power on its virtual PCIe slot using operating system commands. For Oracle Solaris, use hotplug enable. For Oracle Linux, use echo one. For complete procedures, see the Service Manual at: www.oracle.com/goto/x5-4/docs-videos
