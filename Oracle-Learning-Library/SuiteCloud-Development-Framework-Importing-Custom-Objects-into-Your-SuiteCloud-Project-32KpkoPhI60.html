SuiteCloud Development Framework: Importing Custom Objects into Your SuiteCloud Project

2018-05-03
this is SDF in two minutes or less welcome to importing custom objects into your sweet cloud project in this session we will cover the import wizard searching for objects by type and searching for objects by script ID let's get started once you have created your sweet cloud project you can import custom objects from your NetSuite account into the project the XML representation of these objects will be stored under the objects folder right click on the objects folder to display a menu from there under the NetSuite option select import custom object from account select the NetSuite account from which you would like to import these objects remember this list only shows accounts that you already have access to and that are configured in suite cloud IDE click Next to start the import wizard a search assistant lets you find the custom objects that you want to import you can search by the script ID or a portion of it by entering the keyword in the script ID field click search to retrieve the list of objects that contain the keyword as part of its script ID you can also search for custom object by object type click the add/remove button and you will be presented with a list of available object types you can choose all of them or specific ones by clicking on the name move the object to the list on the right side using the navigation buttons click search and a list of objects will appear at the bottom of the screen check the ones that you want to import when you finish completing your selection click finish and the objects will be imported into your project simple like SDF thanks for watching
