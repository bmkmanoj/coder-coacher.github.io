Getting Started with Conversations in Oracle Content and Experience Cloud

2017-12-01
hi and welcome with Oracle content and experience cloud you can share your comments about folders and files to hold a conversation with your fellow collaborators it doesn't matter whether you are using your mobile device desktop or a web browser to get started in your browser select the file or folder you want to discuss and use the comment pane to start a new conversation or post a message to an existing one the message is visible to anyone who shares the folder with you they can reply to the comment or like the comment letting you create a dialogue about a file or folder accessible from any device for example if you have the desktop sync client installed a radio menu is added to your Microsoft Office applications a quick way to access the conversation of a file you're working with on your desktop is to click the conversation icon in the radial menu or right-click on a sync file or folder and select Oracle content than conversation any comments you add here are synced to the cloud and available to everyone with access to the conversation join the conversation of an item on your mobile device by tapping the menu icon next to an item and select conversation if you want to comment on a specific section of a document use your browser to add an annotation to an existing conversation annotations are listed as a comment in the conversation pane view the file click the annotation icon then click on the specific section or click and drag to highlight an area enter your comment to help others see exactly what you're talking about and click an annotation pin to highlight the associated comment in the comment pane you can see a list of conversations you are part of under the conversation tab from here you can also begin a conversation that isn't associated with any folder or file thanks for watching for more information visit cloud.oracle.com
