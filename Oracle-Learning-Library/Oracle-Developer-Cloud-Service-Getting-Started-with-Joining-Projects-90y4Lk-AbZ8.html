Oracle Developer Cloud Service: Getting Started with Joining Projects

2015-03-11
an Oracle developer cloud service project is a collection of tools and services that help developers share resources and project information when they collaborate on a project each project can have one maven repository and multiple git repositories for managing the source code each project also comes with an issue tracker a build server and wiki server for internal documentation the first time that you log into Oracle developer cloud service you'll want to select a project if you're invited to be a project member the project will be listed under the member tab a list of all the shared projects any organization are displayed under the all tab some projects in the organization are private and only project members can see them each project has a home page that displays an overview of the activity of the project and the location of the source code repositories the URL of the remote git repository that you use when you clone the repository is visible in the repositories tab each project also has an issues page it displays a list of project issues and has a link for submitting new issues project members receive notifications when an issue is assigned to them notification settings are set in the preference page available from the user menu select help in the user menu to see the documentation for Oracle developer cloud service
