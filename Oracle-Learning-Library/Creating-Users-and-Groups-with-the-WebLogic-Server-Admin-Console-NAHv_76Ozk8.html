server administration console first I launch the admin console in a web browser by entering the URL for the administration servers host and port followed by slash console then I log in with my admin level username and password once logged in I'll zoom in a little so things are easier to see on the left in the domain structure area I find and click security realms when the security realms page appears I click on the name of the realm in this case my realm next I click the users and groups tab then ensure the users sub tab is selected to create a new user I click new why didn't I have to lock and edit first creating a user or group is not a change to the domains configuration so there is no need to lock it creating a new user is just using the admin console to communicate with the authentication provider on the create a new user page I enter the user's name in this case buck Russell and if desired the optional description then I select the authentication provider in my domain there's only one then I enter the password twice to confirm it once in the password field and then in the confirm password field finally I click OK other users may be added in this way as well now let's create a new group I'll click the group's tab under users and groups then click new give the group name an optional description choose the provider if there's more than one and then click OK once I've entered users and groups I want to make sure users are in their proper groups I click the users sub tab again and click the user of interest in this case buck Russell in the settings for buck click the group's tab select the group to move from available to chosen I choose the employees group click the arrow then click Save now book is part of the employees group that's how to create
