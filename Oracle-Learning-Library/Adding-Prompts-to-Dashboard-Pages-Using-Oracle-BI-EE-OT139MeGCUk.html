if the analysis does not contain a filter I need to add one I click the criteria tab I review the filters pane I don't see any filters in the filters pane for the prompt to work properly on the dashboard I need to add a filter for brand that uses the is prompted operator adding a filter with the is prompted operator for a filters column flags the column is ready to be filtered by a prompt this means when a prompt is used results include only records where the data in the column that is prompted matches the user's choices I click the create a filter for the current subject area button in the filters pane and then I select brand because I want this column to be driven by the dashboard prompt I select is prompted in the operator list I click OK in the new filter dialog to create the filter I see that the brand filter is added to the filters pane I need to save the updated analysis I click Save next I need to create a dashboard prompt this prompt will be available in the catalog for anyone to use once I have saved it I click new in the global header and then I select dashboard prompt I need to add the prompt to the brand column I click the new button in the toolbar I select column prompt in the select column dialog I navigate to and select brand from the products folder I click OK in the select column dialog I accept all defaults in the new prompt brand dialog and click
