Retail Documentation–ReIM: Creating Merchandise Invoices

2017-08-24
a merchandise invoice is a document that a supplier sends to a retailer for merchandise items a merchandise invoice can also contain additional non merchandise costs because a merchandise invoice must involve items only suppliers can send merchandise invoices the create document screen is launched when you choose to create a document through one of the following methods clicking the create document link under the document maintenance subgroup on the global navigation section or for this documentation video we can click on the tasks icon and see document maintenance click that link and select create document from the Actions menu on the search results pane in the document type field select merchandise invoice once the document type is selected the field is disabled and the vendor type is set to supplier you are now ready to enter in the header information you can enter information in many ways in the vendor field select the vendor against which the invoice is raised in the order number field select the purchase order against which the invoice is raised note if a supplier is specified before specification of the order number the combobox list of values drop down displays orders corresponding to all suppliers in the same supplier group as the supplier specified in the supplier site field select the supplier site against which the invoice is raised in the location field select the invoicing or billing location from the list of valid locations such as stores and physical warehouses these locations are defined in retail merchandising system note the location field value is - faulted if the order number selected has a single location to which it is associated for import purchase orders the import location is used as a location in our eim typically you will have an order number you can click the search icon and then type the order number click search and highlight the order number displayed click OK to return to the previous screen the terms field defaults to the term of the specified pio the defaulted value can be modified on the invoice document in addition the supplier or vendor supplier site and location also default based on the information contained in the PIO in the document field enter the identifier for the document in our eim that is used in communication with the vendors and will be displayed to you this must be unique by vendor document type and optionally depending on system setting by document date in the document date field enter the date the invoice or document was created or click the calendar button and select a date
