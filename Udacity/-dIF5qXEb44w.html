in this lesson we covered a number of ways to blend data together using really different approaches we unioned multiple similar data sets to create one longer data set we joined different files together that had a field in common to create a data set containing fields from both sources we use fuzzy matching to match together two data sets with non identical data and lastly we use spatial matching to blend two sources of data together using spatial components like points and regions as I mentioned in the beginning of the lesson we rarely get the data we need in one file so it's really useful to know how to blend data together from whatever sources you have at your disposal to create your final analytic data set
