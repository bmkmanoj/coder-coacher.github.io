Dependency Structure Matrix

2015-02-23
the next few principals make use of a device called a dependency structure matrix or DSM this device was devised by Baldwin and Clark to deal with general with a management of design changes a DSM is a boolean matrix in which the rows and columns correspond to components with a 1 in a Cell indicating that the component in a given row depends on the component in the given column note that the order of rows and columns doesn't matter to the information conveyed so we can feel free to permute them in order to produce more meaningful views
