Check Your Preferences in Oracle SQL Developer and SQL Developer Data Modeler

2018-03-15
It's time for another 2 Minute Tech Tip! Hello, I'm Heli Helskyaho from Finland. I’m COO for Miracle Finland. And the clock starts now! If you're using Oracle SQL Developer or Oracle SQL Developer Data Modeler make sure you know about Preferences. Preference will change how the tool works and how it looks. So if you don't like something that the tool is doing, check the Preferences. Every version of both of these tools will have new preferences that will change a lot of things. So make sure every time you get a new version, check the Preferences. And every time you don't like what the tool is doing or how it looks, change the Preferences. The preferences are in different categories, like environment, like data model, like logical model, this kind of thing. So different categories, and under those categories you’ll find the actual preferences. Go and change them, especially for DDL generation. I have heard people sometimes don't like that the details look different than you were expecting. So check the DDL preferences. Thank you! Was that a good tip?
