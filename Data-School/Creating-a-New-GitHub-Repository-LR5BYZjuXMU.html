Creating a New GitHub Repository

2014-05-01
In this video, we're going to create a new GitHub repository. To get started, click on the plus sign and then click "New repository". You'll need to choose a name, and optionally a description. Choose "Public", and then let's check the box that says "Initialize this repo with a README". Click "Create", and you're done. Notice that it created a "README.md" file, which we talked about previously in part 1 of the "Navigating a GitHub Repository" video. This repo now exists on GitHub only, not your local computer. In the next video, I'll show you how to copy this repo to your local computer so that you can make changes to it, such as adding or modifying files.
