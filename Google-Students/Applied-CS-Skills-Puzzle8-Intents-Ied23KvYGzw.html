what is an intent when developing your Android apps it's likely that you will want to have more than one screen or activity for your audience to use to start or switch between activities you use a mechanism called an intent and intent tells Android what activity you want to switch to it allows you to pass information to it if you know exactly which activity you want to switch to you can use an explicit intent otherwise you can use an implicit intent information is passed inside the intent by a bundle into which you can put extra information for example if you want to take a picture with the users phone you could create an intent to start the phone's image capture activity and you can put extra data in the bundle to specify the path and name of the file you want to save to or the image quality you want to use once you're ready for the user take a picture you can start the intent after the image capture activity returns an intent you can look at the bundle inside of the intent to find out whether a picture was taken successfully or what the user cancelled be sure to visit the Android developers site for further documentation and examples you
