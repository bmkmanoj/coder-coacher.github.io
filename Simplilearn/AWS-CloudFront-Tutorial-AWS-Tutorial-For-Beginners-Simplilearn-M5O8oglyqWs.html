results in objects traveling shorter distances thereby improving website performance but what about files that are not cached Amazon CloudFront continues to maintain connection with servers that originally host content files so when a visitor requests content Amazon CloudFront ensures that the requested objects reach the visitor immediately Amazon Cloud Front supports users requesting data
