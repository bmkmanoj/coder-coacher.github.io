I've heard about Code to Learn from a teacher because, fortunately enough, I was being a part of various inter-school competitions and they have informed me that there is a competition and if I'm interested, I can take part. For Google Code to Learn contest I've made a game called <i>Make in India</i> which is a classic fighting game. During the period of this contest, I've faced a difficulty that was, on the day of submission my system crashed. I've found coding very interesting and I definitely want to be a programmer and make games because I love to play games and I love to make my <i>own</i> games. I would encourage others to participate in the Code to Learn contest because they provide you a platform wherein you just need to drag-and-drop, you don't require any coding skills to make applications.
